**Effort:** 30 min avec Docker Compose actif.

#### 4.2 History pruning — `orchestrator/src/context/builder.rs`
**Problème:** `take(6)` conserve naïvement les 6 derniers messages, sans tenir compte de la taille des tokens. Une longue conversation peut dépasser le contexte LLM.  
**Solution:** Stratégie token-budget :
```rust
// Heuristique : 4 chars ≈ 1 token
// Budget : ~3000 tokens → ~12000 chars
const HISTORY_CHAR_BUDGET: usize = 12_000;
```
Garder les messages les plus récents tant que la somme des longueurs < budget.  
**Effort:** ~1h.
